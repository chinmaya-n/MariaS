Edubot 1.0

Worlds First Comprehensive Educational Robot

             Marias Edubot 1.0 in being recognized over the world to be the first comprehensive Edubot, which explains physics concepts crystal clear and the media is calling it a “Portable Working Laboratory”. Indeed it is !

             The product consists of a robot which in itself explains about 40 concepts and has provisions for attachable-detachable panels. So the Edubot 1.0 plus the 7 panels explain all the physics concepts from 6th grate to 10th grade. It covers all the topics in CBSE (NCERT) syllabus and Andhra Pradesh state prescribed text book syllabus.)

             For Eg:- The robot has an ammeters and a voltmeter as the symbolic eyes(of the robot). Now to explain the force being applied on an object to climb up an inclined plane is W = FS COSθ and work done = V x I x T so VIT=FS COS0. So F = VIT/(S COSθ), can be easily calculated. The concepts of friction, eg. are of motion are also explained lucidly.

             It in a one-of-its-kind product for schools. The product has been made interesting, effecting and portable. It also provides great value for monk.
